# Myth n Wonder

## Overview
A new business venture website for the Erbs launching in 2026. This is a simple landing page built with Express.js serving static HTML/CSS.

## Project Structure
```
/
├── server.js          # Express server (port 5000)
├── public/
│   ├── index.html     # Main landing page
│   └── styles.css     # Styling
├── package.json       # Node.js dependencies
└── README.md          # Project info
```

## Tech Stack
- **Runtime**: Node.js 20
- **Framework**: Express.js
- **Frontend**: Static HTML/CSS

## Running the Project
The server runs on port 5000 using Express to serve static files from the `public` directory.

```bash
npm start
```

## Development Notes
- Server binds to 0.0.0.0:5000 for Replit compatibility
- Static files served from `/public` directory
